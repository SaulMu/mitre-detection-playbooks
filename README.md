# MITRE Detection Playbooks

This repository contains detection playbooks mapped to MITRE ATT&CK techniques. Each playbook includes technique context, relevant logs, detection ideas, investigation steps, false positive considerations and response recommendations.

## Techniques Covered

| Technique | Name | Tactic |
|---|---|---|
| T1059.001 | PowerShell | Execution |
| T1110 | Brute Force | Credential Access |
| T1078 | Valid Accounts | Defense Evasion / Persistence / Privilege Escalation / Initial Access |
| T1021 | Remote Services | Lateral Movement |
| T1003 | Credential Dumping | Credential Access |
| T1566 | Phishing | Initial Access |

## Goal

The goal is to build practical detection engineering knowledge by translating attacker behaviors into actionable SOC detections.
